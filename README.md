# openclaw GitHub Pages package

把這個資料夾內的檔案放進 GitHub repo `openclaw` 的根目錄，然後到 GitHub Pages 設定：

- Source: Deploy from a branch
- Branch: main
- Folder: /(root)

上線網址預期為：

- https://skyline19800302-netizen.github.io/openclaw/

## 檔案
- `index.html`：大阪核帳互動頁
- `.nojekyll`：避免 GitHub Pages 套 Jekyll 處理

## 備註
- 頁面內建 JPY -> TWD 匯率 0.2028
- 可勾選項目、填寫認領人、即時計算總額
